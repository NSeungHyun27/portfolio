UNREAL ENGINE

개요

Unreal Engine으로 프로토타입부터 실제 프로젝트까지 참여했습니다. 블루프린트와 C++를 조합한 게임플레이 구현, 애니메이션·UI·멀티플레이어 기능을 다뤘습니다.

역할 및 성과

  • 캐릭터 컨트롤러, 입력 바인딩, 애니메이션 블루프린트 연동
  • 위젯(UMG)을 이용한 메인 메뉴·HUD·설정 UI 제작
  • C++ 게임플레이 프레임워크(Actor, Component) 확장
  • 멀티플레이어 테스트 및 리슨 서버/클라이언트 구조 이해

기술 스택

Unreal Engine 4/5 · Blueprint · C++ · UMG · Animation Blueprint
